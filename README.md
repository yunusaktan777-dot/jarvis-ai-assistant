# J.A.R.V.I.S - AI Assistant
## Just A Rather Very Intelligent System

Gelişmiş sesli kontrol özelliğine sahip, ChatGPT tarafından desteklenen profesyonel bir yapay zeka asistanı.

### 🎯 Temel Özellikler

- 🎙️ **Sesli Komut Tanıma** - Doğal dili anlama ve işleme
- 💬 **ChatGPT Entegrasyonu** - Akıllı konuşmalar ve cevaplar
- 🔊 **Sesli Çıkış** - Metin-to-Speech teknolojisi
- 🖥️ **Sistem Kontrolü** - Bilgisayar komutlarını yürütme
- 🎨 **Modern Arayüz** - Profesyonel PyQt6 tasarımı
- ⚙️ **Ayarlar Yönetimi** - API key ve kişiselleştirme
- 📊 **Komut Geçmişi** - Tüm işlemleri kaydetme
- 🚀 **Otomatik Başlatma** - Sistem başlangıcında aktif

### 📋 Sistem Gereksinimleri

- Python 3.9+
- Windows 10+ / macOS / Linux
- 2GB RAM
- İnternet bağlantısı

### 🛠️ Kurulum

```bash
# Repository'yi klonlayın
git clone https://github.com/yunusaktan777-dot/jarvis-ai-assistant.git
cd jarvis-ai-assistant

# Virtual environment oluşturun
python -m venv venv

# Aktivasyon
# Windows:
venv\Scripts\activate
# macOS/Linux:
source venv/bin/activate

# Gerekli paketleri yükleyin
pip install -r requirements.txt
```

### 🚀 Başlatma

```bash
python main.py
```

### ⚙️ Yapılandırma

1. Uygulamayı çalıştırın
2. Ayarlar sekmesine gidin
3. OpenAI API Key'inizi girin: [https://platform.openai.com/api-keys](https://platform.openai.com/api-keys)
4. Kaydet ve yeniden başlat

### 🎤 Kullanım

**Sesli Komut:**
- Mikrofon simgesine basın veya `CTRL + ALT + J` kısayolunu kullanın
- Komutunuzu söyleyin
- JARVIS otomatik olarak yanıt verecek

**Metin Komut:**
- Metin kutusuna yazın ve Enter'a basın
- JARVIS yanıtını gösterecek ve sesli olarak okuyacak

### 📝 Örnek Komutlar

- "Merhaba JARVIS"
- "Saat kaç?"
- "Bilgisayarım hakkında bilgi ver"
- "Bir şarkı başlat"
- "Dosya yöneticisini aç"
- "Hava durumu nasıl?"
